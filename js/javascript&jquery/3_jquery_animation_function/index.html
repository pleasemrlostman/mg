[ jQuery 애니메이션 관련 메서드 ]

  1. $("대상").fadeIn(); ->서서히 나타날 때 숨겨져 있는 요소가 서서히 보일 때
  2. $("대상").fadeOut(); ->보이는 요소를 서서히 사라지게 할 때
  3. $("대상").fadeToggle(); ->숨김-나타나기 나타나기-숨김
  4. $("대상").slideUp();->선택된 요소의 높이를 0으로 만들어 설정
  5. $("대상").slideDown();->요소의 높이를 처음에 설정된 높이만큼 아래로 애니메이션
  6. $("대상").slideToggle();->클릭하면->높이0, 또 한번 클릭하면 설정값으로
