jQuery란?

제이쿼리는 자바스크립트의 생산성을 향상 시켜주는 자바스크립트 라이브러리입니다. 자바스크립트의 필요한 함수를 모아둔 곳 원하는 함수를 찾아서 사용하는 방식

jQuery특징

  1. 엘리먼트를 선택하는 강력한 기능 제공
  2. 선택된 엘리먼트들을 제어할 수 있는 다양한 함수
  3. 제공된 자바스크립트를 라이브러리화 함

jQuery문법

  1. $("대상").매서드();

jQuery 랩퍼?(Wrapper)

  1. $(document).ready(function(){})
  2. $(function(){})

jQuery 부호($)의 역할

jquery $(의) 역할은 가로안에 넣을 요소를 선택할 때 사용한다

jQuery 부호($)에 들어가는 이름은?

  1. CSS선택자
  2. HTML마크업 태그
  3. 자바스크립트 객체

jQuery 문서 연결방법

  1. jQuery 문서 연결방법 직접연결시 : 제이퀘리 공식사이트에서 라이브러리를 다운받아서 직접 문서에 연결하여 서비스함
  2. CDN 주소로 연결시 : 서비스를 제공하는 업체로부터 링크주소를 사용해서 문서에 연결하여 서비스 함.
