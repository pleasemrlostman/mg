[ 자바스크립트 변수란 ]

스크립트는 자신의 작업을 시작하는동안 어떤 정보들을 임시로 저장해야할 필요가있다. 이때 스크립트가 데이터를 저장해 두는 것을 변수라고 한다.

1.자바스크립트 변수의 종류

  1. 전역변수 : 문서 전체에 영향을 주는 변수로, 영역(함수)에 관계없이 어디서든지 사용이 가능한 함수
  2. 지역변수 : 함수내에서 사용되는 변수로 그 함수 내에서만 영향력을 가지고 함수롤 호출했을 경우만 작동하다가 함수가 종료되면 소멸되는 변수를 징겨변수라고 한다

2.변수선언방법

  1. 변수는 반드시 옆문 대소문자가 와야한다 (소문자쓰는걸 권장)
  2. 변수명 첫 글자는 숫자가 오면 안된다.
  3. 변수명 사이에 공백을 줄 수 없다.
  4. 변수명은 영문 대소문자를 구분한다
  5. 변수명으로 색상이 변하는 예약어는 사용할 수 없다.(자바스크립트 기본언어)
  6. 변수명에 특수문자를 사용할 수 없다.
  7. 변수명은 의미를 담을 수 있는 이름으로 지정한다
